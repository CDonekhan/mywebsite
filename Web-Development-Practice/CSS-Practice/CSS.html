Set-up for CSS!

Default Stylesheet

There is a default stylesheet that browsers (chrome, firefox, safari, etc) use to display html documents with no css. Additions of css will modify this stylesheet.

If you right click an aspect of a webpage (line of text or headline) and click inspect, the browser will show you all the stylings that aspect contains. You can try it on this webpage!

Adding CSS to Page!

An inline style adds the css directly into a line of html code using a style="" attribute!

html code:

<p style="color:lightblue;">This paragraph should be lightblue!</p>

output:

This paragraph should be lightblue!

An internal style sheet uses a <style> markup tag inside of the <head> tag to add styling to multiple parts of the page at once!

html code:

<head>
    <style> 
        p {
            color:lightblue;

        }
    </style>
</head>
<body>
    <p>This paragraph text should be lightblue</p>
    <p>This paragraph text should also be lightblue since that rule applies to all paragraphs</p>
</body>

output:

This paragraph text should be lightblue

This paragraph text should also be lightblue since that rule applies to all paragraphs

An external stylesheet uses a <link> tag inside the <head> section to link to an external .css file. The <link> tag uses the href"" tag to provide the path to the .css file and the rel="" attribute to register the .css file as a stylesheet for the webpage!

html code:

<head>
    <link rel="stylesheet" href="CSS.css">
</head>
<body>
    <p>This is a paragraph that can now be affected by the CSS.css stylesheet!</p>

</body>

css code inside CSS.css file:

p {
    color:lime;

}

output:

This is a paragraph that can now be affected by the CSS.css stylesheet!

The priority of styles added to a page is inline styles > internal styles > external styles. This means that the page will use the inline styles before the internal styles when loading the page.

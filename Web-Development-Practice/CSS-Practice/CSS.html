Set-up for CSS!

Default Stylesheet

There is a default stylesheet that browsers (chrome, firefox, safari, etc) use to display html documents with no css. Additions of css will modify this stylesheet.

If you right click an aspect of a webpage (line of text or headline) and click inspect, the browser will show you all the stylings that aspect contains. You can try it on this webpage!

Adding CSS to Page!

An inline style adds the css directly into a line of html code using a <style> attribute!

An internal style sheet uses a <style> markup tag inside of the <head> tag to add styling to multiple parts of the page at once!

An external stylesheet uses a <link> tag inside the <head> section to link to an external .css file. The <link> tag uses the href"" tag to provide the path to the .css file and the rel="" attribute to register the .css file as a stylesheet for the webpage!

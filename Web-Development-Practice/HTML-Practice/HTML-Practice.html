opening-tag: <p>
content: Hello World
closing-tag: </p>

Webpage Structure

General Page Layout

To begin a webpage a <!DOCTYPE html> tag is written to specify the html type of the page. The page is then wrapped in an <html> tag to display where the html content is.

                                                  <!DOCTYPE html>
                                                    <html>
                                                      Rest of document bruh
                                                    </html>

The webpage is then sectioned into the <head> and the <body>. 

                                                    <!DOCTYPE html>
                                                      <html>

                                                        <head>
                                                          Content of the head
                                                        </head>

                                                        <body>
                                                          content of the body
                                                        </body>

                                                      </html>

The Head

The <head> contains information that the webpage uses and references such as <style> tag for css files and <script> tag for java files. We will focus only on the css for now!

html is used to create the structure of the page while css is used to style the page by: adding colors, changing fonts, changing border sizes, and much more! css is a language that will be covered more in depth in the css section but it will be utilized in small amounts in this section! The main thing to know now is that css can be added into an html file either directly or through a link. 

                                                      <!DOCTYPE html>
                                                        <html>

                                                          <head>
                                                          <title> This is the title of the document! </title>
                                                          <style>

                                                                html {
                                                                  color:red;
                                                                }

                                                            </style>

                                                            <link rel="stylesheet" href="sample.css">

                                                          </head>

                                                        <body>
                                                          (content of the body)
                                                        </body>

                                                      </html>

As you can see, a <style> tag is used to add css directly into the html to change the color of text in the document to red. A <link> can be used to associate a css page to the html document. The href attribute is used to designate the relative location of the css file. The rel="stylesheet" attribute confirms that the <link> recognizes the document as a styling reference. The red color code can be added to the css page to affect this html document in the same way!

You may have notices the <title> tag in the <head> section! This tag gives a title to the document that is used by search engines and the favorites tab of a web browser to identify this page.

The Body

The <body> is where all the content of the webpage is displayed.

The <body> is typically sectioned into <header>, <footer>, and the rest of the content. The <header> and <footer> usually display content that is identical to all webpages in the website. The <header> will have the website title, some navigation links, and maybe a small description. The <footer> typically has company contact information, copyright information, and other navigation elements. The following example is the html code for the <header> and the <footer> of this page!

Many of the tags may be unfamiliar but they will be explained later in this document. The main takeaway is the existance of a <header> and <footer>. 
<body>
      <header>
         <h1>HTML Practice</h1>
          <nav>
                  <a href="../../index.html"><img class= "FPic2"src="../../Index-Resources/home.png" 
                  alt="Link to Home Page" title="Home Page"></a>

                  <a href="../../Navigation-Pages/Resume-Page/Resume.html"><img class= "FPic2" src="../../Index-Resources/resume.jpg"
                  alt="Link to Resume" title="Resume"></a>

                  <a href="../../Navigation-Pages/Contact/Contact.html"><img class= "FPic2" src="../../Index-Resources/social.png" 
                  alt="Link to Social Media Links" title="Social Media Links">   </a>

          </nav>

      </header>
           Rest of the Document!
      <footer>
         <p><a href="#top"> Go to the top!</a></p>
         <p>© <small>2021 CDonekhan LLC</small></p>
      </footer>
</body>

The rest of the <body> is typically divided into <section> (s) and other markups to organize the page. It is important to section and label portions of the page with the appropriate tags so that people can read the code, search engines can locate information easily, and people who use auditory aids can understand the webpage properly!

Finally, a <main> tag is used to display where the main content of the page is. This content includes parts of the page that are unique only to the webpage. For example: A webpage with a news story will have the <main> tag around the content of the news article.

Paragraphs/Ordered Lists/Unordered Lists/Descriptive List

The <p> (paragraph) tag creates lines of text that are separated/spaced out to provide individual paragraphs. The <ol> (Ordered list) tag and the <ul>(unordered list) tag display content in a list format. The <ul> (Unordered list) display dots by default and an <ol> (Ordered list) will display list items with numbers. A <dl> (descriptive list) tag gives titles and content for each item on the list.

Links/Navigation

The <a> tag is used to create a hyperlink of the content it wraps. The <a> is called an anchor. It uses the href="" attribute to specify the path to the new webpage or section on the current webpage. The target attribute of the <anchor> tag can designate if the link is opened in a separate tab.

This section has a <nav> tag (Navigation)! <nav> is used to designate locations where multiple links to other websites are listed such as contact info or links to other parts of the website!

There is multiple ways to designate the location of a webpage using the href="" attribute:

  1. A url can be used for a link to a webpage.
  2. The directory can be used to direct from the current folder to the webpage being linked to.
  3. The root directory is similar to the directory, except the beginning of the path starts with the root folder of the website and moves down. Root directory links can only be used on a live website since only online servers have a root folder.

There is also an example of navigation within a webpage in the navigation links at the top of this page! html for this link: <a href="#top-navigation">top of this page!</a>

There is also mailto: command that can be added to an href="" attribute to send an email. The ?subject= aspect of the mailto: command adds a title to the email being sent! (%20 is code for space)!

Examples of these concepts:

html code:

<nav>
    <p><a href="https://www.youtube.com/watch?v=3egXqihisFQ">This is a special link that redirects to another page!</a></p>

    <p><a href="https://www.youtube.com/watch?v=D2_r4q2imnQ" target="_blank">This is a special link that redirects to another page on another tab!</a></p>

    <p><a href="https://cdonekhan.github.io/mywebsite/Just-For-Fun/Buttons/Buttons.html">Url for buttons using direct link</a></p>

Address/Break/Strong Tag/Small Tag

An <address> tag is for a geographical address. The <address> tag outputs text in italics and can be found in a search engine as an address! The <strong> tag is used to indicate importance or urgency. The <strong> tag outputs bold text by default. A <br> tag is a break. The break places information in separate lines for the same text section. The <small> tag is used for disclaimers, caveots, legal restrictions, and copyright information. the <small> outputs text at a smaller font.

An example of these concepts:

html code:

<h2>Contact</h2>
    <p><strong>Email:</strong> us at coolrobinex@gmail.com</p>
    <p><strong>Phone:</strong> 555-555-5555</p>
    <p><strong>Address:</strong></p>
    <address>
        Robin Expertise <br>
        5555 Robin Way<br>
        City, State 33333
        <p><small>*There are currently no in person visits at the moment. Please call for information on shipping!</small></p>
    </address>

Contact

Email: us at coolrobinex@gmail.com

Phone: 555-555-5555

Address:

Robin Expertise
5555 Robin Way
City, State 33333

*There are currently no in person visits at the moment. Please call for information on shipping!

Horizontal Rule/Emphisis

An <hr> tag is used to separate different blocks of content inside of a <section>, <article>, etc. This separatation is made for thematic breaks and subject (concept) changes. <em>(emphisis) tags add emphisis to particular words to change the meaning of the sentance. <em>> outputs text in italics!

Examples of these concepts:

html code:

            <p>The first rule of Fight Club is: You <em>do not</em> talk about Fight Club. I don't think I have to explain why this rule is so important!</p>
            <hr>
            <p>The second rule of Fight Club is: <em>Always</em> bring cupcakes. <em>always bring cupcakes.</em></p>

Output:

The first rule of Fight Club is: You do not talk about Fight Club. I don't think I have to explain why this rule is so important!

The second rule of Fight Club is: Always bring cupcakes. always bring cupcakes.

span/div vs pre

<span> and <div> are markup tags that place non-specific labels on blocks of text and the labels can later be edited in css. <span> is used inside a block of content (such as a word or phrase in a sentence), while <div> is used for an entire block of content (an entire paragraph is surrounded by dev).

<pre> is a markup tag that displays text & spacing exactly as it appears in the html.

Webpage Structure

General Page Layout

To begin a webpage a !DOCTYPE html is written to specify the html type of the page. The page is then wrapped in an html tag to display where the html content is.

The webpage is sectioned into the head and the body.

The head

The head contains information that the webpage uses and references such as css files for style and script for java. The title is added in the head for the page name in favorites/bookmarks. There are also different formatting options usable in the head section.

The Body

The body is where all the content of the webpage is displayed.

The body is typically sectioned into header, footer, and the rest of the content. The header and footer usually display content that is identical to all webpages in the website. The header will have the website title, some navigation links, and maybe a small description. The footer typically has company contact information, copyright information, and other navigation elements.

The rest of the body is typically divided into section (s) and other markups to organize the page. It is important to section and label portions of the page with the appropriate tags so that people can read the code, search engines can locate information easily, and people who use auditory aids can understand the webpage properly!

Finally, a main tag is used to display where the main content of the page is. This content is usually parts of the page that are unique only to it, like a page that displays an article. The article is the main content.

Classes/Images

A class is added to a markup/tag to designate style/css to be added to that line of content. The style changes to a partiuclar class can be found in the css stylesheet attached to the html document. The stylesheet can be found in the head section.

An img tag (Image) is used to add an image to the webpage. An image can have three attributes: the src atttribute adds the link to the image source; the alt attribute adds text information used to label the image in the html and when the image source is compromised; and the title attribute, which is displays a text description of the image if a cursor hovers over the image.
